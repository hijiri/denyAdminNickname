●第3者に管理者ニックネームでのコメント投稿を禁止するプラグイン

■概略
このソフトウェアは、Loggixでコメントを投稿する際に、管理者ニックネームを使用した場合はパスワードを要求するプラグインです。

■詳細
Loggixでは標準で、訪問者でも管理者ニックネームを使用しようしてコメント投稿する事が可能な為、第3者に管理者になりすましてコメントを投稿されてしまう可能性があります。

このプラグインでは、管理者ニックネームで投稿する場合、対応する管理者パスワードで認証を行い様にして、なりすましを防止します。

■インストール/アンインストール方法
インストール
    /plugins/へdenyAdminNickname.phpをアップロードします。
アンインストール
    /plugins/からdenyAdminNickname.phpを削除します。

■使用方法
管理者ニックネームで投稿する際は、コメント投稿フォームの「編集用パスワード」にニックネームに対応する管理者のパスワードを入力してください。

■その他
Loggixでは複数の管理者で同じニックネームの使用が許可されている為、このプラグインでも、同じニックネームを使用している管理者のパスワードであれば、どれを使用しても認証を通過する事になります。

例：ニックネーム"hogehoge"を使用する管理者は"foo"と"bar"の2人の場合。
　　"hogehoge"でコメントを投稿する場合は"foo"と"bar"どちらのパスワードでも投稿出来る。
